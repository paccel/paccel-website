Customizable Arrays

Paccel produces customized array for the required size and aspect ratio. It integrates IP blocks to address wide variety of applications across wireless, compute, storage markets. Its logic tiles embed required RAM/FIFOs, DSP, FPU, neural network nodes or other market specific application blocks using uniform architecture and flow. These flexible customized arrays can offer 20-100x performance benefit over software solutions.

Proven silicon and software tools

Paccel provides silicon proven IP blocks as a soft RTL or hard GDS drop. PCC software tool reads synthesized designs and runs map, place and route to produce configuration memory values. Appropriate 3rd party tool support is provided for synthesis as well as functional and timing verification. Built on decades of processor and FPGA experience, architecture and tools are developed grounds up to support the tiling and customization requirements.
