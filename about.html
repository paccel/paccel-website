Introduction

Paccel provides Programmable Logic Accelerator using homegrown FPGA compiler technology. It provides silicon IP and software tools to embed customized FPGA blocks in System on Chip designs. Its patent pending technology enables standard SoC integration and design flows while providing FPGA type flexibility with high performance. Built on decades of processor and FPGA experience, it supports the required logic tile array size and shape. It integrates IP blocks to address wide variety of applications across wireless, compute, storage markets. Its logic tiles embed required RAM/FIFOs, DSP, FPU, neural network nodes or other market specific application blocks using uniform architecture and flow. These flexible customized arrays can offer 20-100x performance benefit over software solutions. Please send email at fpgacompiler@gmail.com and visit http:://www.paccel.com for more information.
